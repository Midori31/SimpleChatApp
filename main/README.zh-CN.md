# 简易图形化聊天应用

一个基于 TCP 协议的轻量型聊天应用，采用 PyQt6 图形界面，支持群聊、私聊及在线状态通知功能。

## 核心功能

- 图形化登录界面
- 群聊消息实时同步
- 私聊功能（格式：`@用户名 消息内容`）
- 在线用户列表自动更新
- 用户上下线通知
- 优雅退出机制

## 技术栈

- Python 3.8+
- PyQt6（图形界面）
- TCP Socket（网络通信）
- 多线程（并发处理）

## 快速开始

### 1. 安装依赖

```Bash
pip install pyqt6
```

### 2. 启动服务器

```Bash
python server.py
```

### 3. 启动客户端（支持多开）

#### 图形化客户端

```Bash
python client_gui.py
```

#### 命令行客户端（可选）

```Bash
python client.py
```

### 4. 使用说明

1. 输入用户名登录（不可重复）
2. 群聊：直接输入消息发送即可
3. 私聊：使用格式 `@目标用户名 你的消息`
4. 退出：输入 `exit` 或直接关闭窗口

## 注意事项

- 服务器默认端口：9000（需修改可编辑 `server.py`）

- 需手动将 `client.py` 和 `client_gui.py` 中的 `SERVER_IP` 更新为服务器所在地址

- 本机测试时，可将 `SERVER_IP` 设置为 `127.0.0.1`

- 确保服务器与客户端处于同一网络环境

- 防火墙可能需要放行对应端口以允许网络访问
